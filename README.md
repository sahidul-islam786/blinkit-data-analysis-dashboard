📊 Blinkit Sales Performance Dashboard (Excel)
📘 Project Overview

The Blinkit Sales Performance Dashboard is an Excel-based data analytics project focused on analyzing sales data to uncover revenue trends, product performance insights, and outlet-level comparisons.

The dashboard transforms raw sales data into interactive visual insights using advanced Excel features, enabling data-driven business decisions.

🎯 Objective

Analyze sales and product-level data

Identify key KPIs such as Total Sales, Category Performance, and Outlet Performance

Build an interactive dashboard for quick performance monitoring

🛠 Tools & Techniques

Microsoft Excel

Pivot Tables & Pivot Charts

Slicers & Interactive Filters

Conditional Formatting

Functions: IF, SUMIFS, COUNTIFS, VLOOKUP/XLOOKUP, INDEX-MATCH

📈 Key Features

Dynamic KPI tracking (Total Sales, Revenue Distribution, Product Ratings)

Category-wise and Outlet-wise performance analysis

Interactive dashboard with slicers for easy filtering

Cleaned and structured dataset for accurate reporting

🔍 Key Insights

A small group of product categories contributes to the majority of revenue.

Sales performance varies significantly across outlet sizes and locations.

Higher-rated products tend to generate more consistent sales.

💡 Business Recommendations

Focus marketing efforts on top-performing categories.

Improve low-performing outlets with targeted strategies.

Continuously monitor KPIs using dashboard tracking for better decision-making.
